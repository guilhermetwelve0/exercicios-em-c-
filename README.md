# exercicios-em-c-

1) Escreva um algoritmo em C++ que imprima todos os números inteiros de 0 a 50.
2) Escreva um algoritmo em C++ que imprima todos os números inteiros de 100 a 1 (em ordem decrescente).
3) Escreva um algoritmo em C++ que imprima todos os números inteiros de 100 a 200. 
4) Escreva um algoritmo em C++ que imprima todos os números múltiplos de 5, no intervalo fechado de 1 a 500.
5) Escreva um algoritmo em C++ que imprima todos os números pares do intervalo fechado de 1 a 100.
6) Escreva um algoritmo em C++ que imprima os 100 primeiros números ímpares.
7) Escreva um algoritmo em C++ que imprima o quadrado dos números no intervalo de 1 a 20.

exercicios propostos e resolvidos em for do e do while
